Robot testing interface

Key points

  • by default - you use created robot named 'lucky'
  • use console.log(lucky) to see it
  • to create new robot use command:
    const robot = new Robot('robot_name')
  • robot is an object
  • to use robot functions - call robot's methods robot_name.method() and watch result in output console
